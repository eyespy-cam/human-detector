# backend

It's the backend.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# bdd tests
$ npm run test:bdd

# test coverage
$ npm run test:cov
```
